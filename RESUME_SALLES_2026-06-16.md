# Kodak Monitor — Nouveauté du 16 juin 2026

## Ce qui change pour vous

Les PC affichés dans les salles/galeries sont maintenant protégés par un
**écran de verrouillage automatique**, pour éviter que le public ne touche
aux réglages quand aucun opérateur n'est devant le poste.

## Comment ça marche

- Si personne ne touche l'écran pendant quelques minutes, un écran noir
  avec un cadenas 🔒 et un clavier numérique apparaît.
- Pour le faire disparaître, il suffit de **taper le code PIN** sur le
  clavier affiché à l'écran (pas besoin de clavier physique).
- Une fois le bon code saisi, l'écran revient normalement.

## Si vous ne connaissez pas le code

Demandez-le à la personne qui gère le poste (chaque PC peut avoir son
propre code). Sans code défini, le code par défaut est **1234**.

## En cas de souci

- Si l'écran reste bloqué ou si le code ne fonctionne pas, contactez
  le support technique plutôt que de redémarrer le PC.
- Ce verrouillage protège uniquement l'écran/le logiciel — il n'empêche
  pas l'imprimante de fonctionner normalement pendant ce temps.
